Accessibility Report

After the initial scan using the achecker online tool, the below problems were reported, and the following report was generated.

  • Input element missing an associated label. As the inputs don't have a label, a title was added instead.
  • Incorrect usage of the i tag for icons A span was used instead.
  • Useless links. Some of the links did not have a target url.
  • Header nesting. Incorrect usage of successive headers h6 after h4, h6 was removed

The above issues were resolved and a new report was generated, which passed with flying colours.
